# Demo-Swagger2-Spring-Boot
demo for using Swagger Specification with spring boot 
run this app on http://localhost:8088/swagger-ui.html to visualize the interface generated by swagger ui 
i have used spring-fox-swagger2 and spring-fox-swagger-ui (the implementation of swagger2 and swagger ui by spring-fox) https://springfox.github.io/springfox/docs/snapshot/
